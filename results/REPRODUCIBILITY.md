## Results Generation Commands

### Step 1: Install dependencies
```bash
cd AgentLayer
pip install -r requirements.txt
pip install -e .
```

### Step 2: Run complete evaluation
```bash
python scripts/run_all_evaluations.py \
    --video <VIDEO_PATH> \
    --backend yolo \
    --output-dir results/ \
    --duration 120
```

### Step 3: Run individual steps (optional)
```bash
# Backend benchmark
python scripts/benchmark_backends.py --video <VIDEO> --backends mock,yolo \
    --output results/benchmark_results.json

# Main evaluation
python scripts/evaluate.py --video <VIDEO> --backend yolo \
    --output results/evaluation_results.json --export-timeline results/timeline.csv

# Baseline comparison
python scripts/compare_baselines.py --video <VIDEO> --backend yolo \
    --output results/comparison_results.json

# Hysteresis ablation
python scripts/ablation_hysteresis.py --video <VIDEO> --backend yolo \
    --output results/ablation_results.json

# Adversarial scenarios (no video needed)
python scripts/adversarial_scenarios.py --output results/adversarial_results.json

# Data export
python scripts/plot_data.py --evaluation results/evaluation_results.json \
    --comparison results/comparison_results.json \
    --ablation results/ablation_results.json \
    --adversarial results/adversarial_results.json \
    --output-dir results/plots/

# LaTeX tables
python scripts/generate_latex_tables.py \
    --evaluation results/evaluation_results.json \
    --comparison results/comparison_results.json \
    --ablation results/ablation_results.json \
    --benchmark results/benchmark_results.json \
    --output-dir results/latex_tables/
```
